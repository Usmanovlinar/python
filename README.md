6.1. Этап 1. База
async def

Это объявление корутинной функции.

async def hello():
    print("hello")

Важно: вызов такой функции не запускает её сразу. Он создаёт coroutine object.

 ```
coro = hello()
print(coro)  # <coroutine object hello at ...>
```

Зачем

Чтобы описывать асинхронную логику: “сделай что-то, потом уступи управление в await, потом продолжи”.

Как это работает

async def создаёт функцию, которая при вызове возвращает объект-корутину. Реальное исполнение начинается, когда ты:

либо делаешь await coro
либо передаёшь корутину в asyncio.run()
либо оборачиваешь в create_task() / T
